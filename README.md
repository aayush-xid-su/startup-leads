# 🚀 400+ Tech & Corporate Companies Career Directory

A complete frontend web project that lists **400+ global companies** with direct links to their official career pages.

This project helps students, job seekers, and professionals quickly access company hiring portals across multiple industries like:

- 💻 Tech
- 🏦 Banking & Finance
- 🧠 AI & Data
- 🏭 Manufacturing
- 🚗 Automotive
- 🎮 Gaming
- 🛍️ E-Commerce
- 🏥 HealthTech
- 📡 Telecom
- ⚡ Energy
- 🏢 Consulting
- And more...

---

## 📌 Project Features

✅ 400+ Companies  
✅ Direct Career Page Links  
✅ Categorized by Industry  
✅ Search Functionality  
✅ Tag-Based Filtering  
✅ Clean & Responsive UI  
✅ Pure HTML + CSS + JavaScript  
✅ No Backend Required  

## 📂 Project Structure
```
project-folder/
│
├── index.html # Main HTML file
├── style.css # Styling file
├── script.js # Company dataset + search logic
└── README.md # Project documentation
```


---

## 🎯 Purpose of This Project

This project was built to:

- Help job seekers quickly explore company career pages
- Avoid manually searching company hiring portals
- Provide a categorized, organized directory
- Serve as a frontend practice project
